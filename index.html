Chrome prototype

Load it locally and test it in Gmail.

The current prototype is an unpacked Chrome extension. It uses only local browser state and Gmail page access.

  1. Open chrome://extensions
  2. Turn on Developer mode
  3. Choose Load unpacked
  4. Select this project folder
  5. Open Gmail and scan the inbox
